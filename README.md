# Документация проекта reboot-the-system

## Введение

Показалось, что просто форма как то скучно, решил добавить пару базовых страниц и какую то дополнительную логику на свой вкус.

Приложнение создал через `npm create vite@latest`, часть конфигов и настроек стащил с других своих проектов.

Использовал Claude Code для:

- Генерации основноно ui для страниц с использованием mui
- Разделения изначального одностраничного приложения на разные страницы и разделение формы на компоненты
- В конце попросивл его добавить lazy loading для роутов, после того как проверил билд

## Структура папок проекта

```
.

```

## Форматирование кода

Форматирование кода в проекте настроено с помощью Prettier. Вам не нужно вручную форматировать код — это произойдет автоматически при коммите или сохранении файла. Желательно настроить автоформатирование по сохранению в вашей IDE.

## Pre-commit хуки

В проекте настроены pre-commit хуки с использованием Husky. При попытке сделать коммит будут автоматически запущены следующие задачи:

- Линтинг закоммиченных файлов
- Полный линт проекта
- Форматирование кода с помощью Prettier

Убедитесь, что все хуки завершаются успешно перед тем, как пушить ваши изменения.

## Как начать локальную разработку

Следуем простой инструкции, чтобы запустить локальную версию и начать работу.

### Требования

- node - для разработки использовать версию `22.13.0` и новее
- npm - следуем инструкциям на [npmjs](https://www.npmjs.com/get-npm)

### Как запустить и собрать локально

1. Клонируем репозиторий
    ```sh
    git clone https://github.com/Nerbeer/reboot-the-system.git
    ```
2. Ставим NPM пакеты
    ```sh
    npm ci
    ```
3. Инициализируем husky для pre-commit хуков (опционально, тк они должны автоматически инициализироваться при установке пакетов)
    ```sh
    npm run prepare
    ```
4. Запускаем дев сборку
    ```sh
    npm run dev
    ```
5. Билдим прод версию
    ```sh
    npm run build
    ```

Дополнительные команды, которые могут понадобиться:

- Полный линт проекта
    ```sh
    npm run lint
    ```
- Запуск prettier по всему проекту
    ```sh
    npm run format
    ```
